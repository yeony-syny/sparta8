과제 해결 순서는 이렇습니다
1) 각 input , textarea, select의 값입력합니다
2) 그리고 버튼을 누르면 위 태그들에 값이 제대로 들어 갔는지 확인하는 절차를 거칩니다
3) 절차를 거치려면 버튼에 함수를 연결해야합니다
4) 연결은 버튼에 onclick 속성에 스크립트단에서 만든 함수를 연결해야합니다


5) 해당 함수엔 값들을 제이쿼리로 가져와 체크하는 로직이 있어야 합니다
6) 해당 로직이 완료되면(정상적으로 모든 태그에 값이 입력된 상태) ajax를 함수 로직 하단에 둡니다

7) 해당 ajax에는 과제에 소개가된 환율 api 주소를 호출하여 success 부분 response에 환율 정보가제대로 담겨왔는지 확인해봅니다
8) 확인은 console.log로 가능합니다
9) 값이 제대로 담긴게 확인이 되면 해당 값에서 환율 값을 변수에 담습니다
10) 환율 값을 담은 변수를 이용해, 작성하시고 계시는 html에 환율 값을 넣을 태그를 마련하여 해당 환율 값을 제이쿼리로 주입합니다
11) 주입은 여러 방법이 있지만 예컨대 이런식으로 가능합니다
12) $(“#exchangerate”).text(환율값 변수)
13) #exchangerate은 html 파일에서 환율 값을 넣을 span태그 정도의 id값입니다
